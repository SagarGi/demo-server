# demo-server

## Setup Locally
1. Clone the project
 ```bash
 git clone https://github.com/SagarGi/demo-server.git
 ```
2. Go to the project directory
  ```bash
  cd demo-server
  ```
3. Install all the packages
```bash
npm install
```
4. seed datdabase
```bash
node ./Database/dbseed.js
```
4. Run the server
```bash
npm start
```
5. visit `http://localhost:3001/ `. The response would be "OK" in the browser
